# Deployment Pipeline

![Deploy Status](https://github.com/koustavx08/deployment-pipeline/actions/workflows/deploy.yml/badge.svg)

This project demonstrates a CI/CD deployment pipeline using GitHub Actions.
Every push to the main branch automatically runs lint checks and deploys the
HTML/JavaScript project to GitHub Pages if all checks pass.

## Features
- Automatic deployment on git push
- ESLint code quality checks
- GitHub Pages hosting
- Zero manual deployment steps

## Technologies Used
- HTML
- JavaScript
- GitHub Actions
- GitHub Pages
- ESLint
